# How to Download .HTML Files from GitHub

To download any of the .html files from this repository:

1. Click on the file you want to download
2. Press **"View Raw"** button (or click on the raw file link)
3. The file will open in your browser - right-click and select **"Save As"** to download
4. Make sure to save it with the `.html` extension
5. Open the downloaded file in your browser to use it
